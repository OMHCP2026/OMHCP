# 🧠 Online Mental Health Counseling Portal (OMHCP)

A web-based platform designed to provide accessible and confidential mental health support to users.

## 📋 About The Project

This portal connects users with professional counselors through a secure and easy-to-use online platform. It provides a safe space for people to seek mental health support without any hesitation.

## ✨ Features

- 🔐 Secure User Registration & Login
- 📅 Online Appointment Booking with Counselors
- 👤 Separate Modules for Users, Counselors & Admins
- 📊 Session & Appointment Record Management
- 🖥️ User-Friendly Responsive Interface

## 🛠️ Technologies Used

|------------|---------|

| CSS3 | Styling & Design |
| JavaScript | Frontend Logic |
